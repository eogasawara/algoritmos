Tutorial de Introdução a Algoritmos

Esta playlist ensina pensamento algorítmico e programação para crianças a partir de 14 anos (ensino médio), usando uma linguagem simples (Portugol), exemplos do dia a dia e desafios em forma de jogos e problemas. Aqui você vai aprender a pensar como um programador: resolver problemas, criar estratégias, usar laços, decisões, vetores e funções para construir soluções inteligentes.
